## 🏁 118. 정수 배열이 너무 좋아
$N$개의 수가 주어질 때, 수들을 오름차순으로 정렬한 결과를 출력하라.

첫 줄에 수의 개수 $N$이 주어진다. (1 ≤ $N$ ≤ 1,000,000)
<br>둘째 줄부터 $N$개 줄에 걸쳐 수가 주어진다. 주어지는 수는 0 이상 1,000,000 이하의 정수이다.

$N$개 줄에 걸쳐, 한 줄에 하나씩 수들을 오름차순으로 정렬한 결과를 출력한다.

### 📝 입력 예제
5<br>
3<br>
1<br>
1<br>
3<br>
4


### 🖨️ 출력 예제
1<br>
1<br>
3<br>
3<br>
4
